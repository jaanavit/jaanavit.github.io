Project 0 — Image Manipulation & Warm-Up

Project 1 — Image Filtering & Hybrid Images

I implemented both single-scale and pyramid-based alignment algorithms to register the red and green channels to the blue channel, using normalized cross-correlation and multi-resolution refinement to handle high-resolution TIFF images efficiently. To improve robustness on challenging images with exposure differences, I incorporated Sobel-based edge maps and an edge-aware scoring function that emphasized structural consistency over raw intensity. The resulting pipeline produced accurate RGB reconstructions across the full dataset, with additional tuning for difficult cases like self_portrait and emir to ensure precise color alignment. Project 2 — Filters and Frequencies

I developed custom 2D convolution implementations—from naive multi-loop versions to optimized vectorized approaches—and validated their correctness against scipy.signal.convolve2d. Building on this foundation, I implemented finite-difference and Derivative-of-Gaussian filters, unsharp masking, hybrid imaging, and multi-resolution (Gaussian/Laplacian pyramid) blending to demonstrate how frequency-domain representations enable precise control over edge structure, detail enhancement, and seamless image compositing.

Project 3 — Image Mosaicing (Manual + Automatic Stitching)

I implemented a full image-stitching pipeline including homography estimation, inverse warping, and multi-scale Laplacian blending to manually create seamless panoramas. I then built an automatic version using Harris corner detection, ANMS, feature descriptors, Lowe’s ratio test, and RANSAC to robustly compute homographies and mosaic images without manual correspondences.

Project 4 — Neural Radiance Fields (NeRF)

I implemented the full NeRF pipeline from scratch, beginning with ArUco-based camera calibration and pose recovery, followed by building a 2D neural field to understand positional encoding and coordinate-based MLPs. I then extended this to a full 3D NeRF with ray generation, stratified sampling, and differentiable volume rendering to reconstruct scenes from multi-view images. Using both the Lego dataset and my own captured object, the model successfully produced coherent novel viewpoints, with performance influenced by near/far plane tuning, learning-rate decay, and network capacity.
